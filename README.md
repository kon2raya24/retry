# retry

[![npm version](https://img.shields.io/npm/v/retry.svg)](https://www.npmjs.com/package/retry)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Tiny async retry utility with exponential backoff

## Installation

```bash
npm install retry
```

## Functions

- `retry` — Retry an async function with exponential backoff
- `retrySync` — Retry a sync function with exponential backoff
